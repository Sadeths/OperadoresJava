# ⚙️ Operadores en Java

Repositorio con ejercicios prácticos enfocados en el uso de **operadores en Java**, aplicados en diferentes escenarios para reforzar la lógica de programación.

---

## 📚 Contenido

Este proyecto incluye ejemplos sobre:

* ➕ Operadores aritméticos
* ⚖️ Operadores de comparación
* 🧠 Operadores lógicos (`AND`, `OR`, `NOT`)
* 🟰 Operadores de asignación
* 🔁 Operadores unarios
* 📊 Precedencia de operadores

---

## 🧩 Aplicaciones prácticas

Los operadores se aplican en ejercicios reales como:

* 🧾 Generación de tickets de venta
* 📐 Cálculo de área de rectángulos
* 🔐 Sistema de autenticación
* 🎯 Validación de rangos
* 💰 Sistema de descuentos VIP
* 📚 Sistema de préstamo de libros

---

## 🚀 Tecnologías utilizadas

* Java (JDK 17+)
